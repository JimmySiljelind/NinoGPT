# chatbot-app

Install Bun on your device:

```bash
irm https://bun.sh/install.ps1 | iex
```

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run dev
```

This project was created using `bun init` in bun v1.2.22. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
